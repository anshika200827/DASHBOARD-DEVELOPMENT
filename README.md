🎬 Netflix Movies & TV Shows — Data Analysis Dashboard


This project performs an exploratory data analysis on a Netflix content dataset containing 49 titles — 26 Movies and 23 TV Shows — released between 1975 and 2021, using Microsoft Excel for data cleaning and preparation and Power BI Desktop for building an interactive dashboard. 
The dataset includes 8 columns covering title, type, director, country, release year, content rating, duration, and genre. Key insights uncovered include that TV-MA is the most dominant rating with 20 titles reflecting a strong mature content presence, the United States leads in content production with 11 titles followed by India and the United Kingdom, and International TV Shows, Dramas, and Thrillers emerged as the top genres across the library. Movies in the dataset average around 100 minutes in runtime while TV Shows range from 1 to 9 seasons, and content production is heavily concentrated around 2020 and 2021 marking Netflix's peak global expansion phase. The Power BI dashboard features interactive slicers for filtering by content type, country, and rating, along with bar charts, pie charts, KPI cards, and a matrix visual with conditional formatting to highlight genre and rating distributions across the dataset. Both the cleaned Excel file and the Power BI report file are included in this repository for full reproducibility of the analysis.
